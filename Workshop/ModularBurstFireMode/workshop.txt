title=Burst Mode
description=This mod adds a simple way to add burst mode to weapons, configurable per weapon instance. You can test it with the scrap guns below, have fun :)
description=
description=[code][h2]< Info >[/h2]
description=Options are saved in the weapon data, this means every weapon can be customized to have it's own settings. If an option is not set, then the default value will be used.
description=
description=This mod works when the weapon is set in FireMode "modBurst". This means you need to add this option to the weapon script.
description=
description=Option Values:
description=modBurst_Shots: Number of shots for the burst. (Default is 3)
description=modBurst_Speed: Speed to use for the shooting animation. (Default is 4, same as Auto fire mode)
description=modBurst_Delay: Applied delay after the burst. (Default is 10)
description=modBurst_AnimFireMode: Used to change the player variable FireMode to determine animation selection. (Default is Auto)
description=
description=not added
description=option to customize whether or not the shots fired counter is reset when you release the trigger.
description=[/code]
description=
